# Resumen de Apuntes - Sistemas Distribuidos I

En este repo hay resúmenes de apuntes de la materia **Sistemas Distribuidos I** de la catedra Espina.

Los resumenes de las clases fueron hecho a partir de la transcripción de las clases y los apuntes hechos durante las mismas, todos los resumenes fueron hechos con ChatGPT y levemente revisados asi que pueden contener errores o ser muy verbosos.

Tambien esta el unico coloquio que se hizo en la materia, con las preguntas y respuestas que se dieron en clase, y a partir de esto y los apuntes ChatGPT hizo preguntas de ejemplo para practicar usando.
